# Stoic-gym

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `data-source.ts` file

- you can use docker:
  `docker compose up -d` (to stop `docker compose down`)
- or you can add your postgres database settings:
  - username: '',
  - password: '',
  - database: '',

3. Run `npm start` command
